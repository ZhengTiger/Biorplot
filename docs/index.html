Chapter 1 Installation & Usage

You can download the package from Biorplot GitHub Page:

git clone https://github.com/zhenghu159/Biorplot.git

Then load the package by:

source('Biorplot.R')

1.1 Dependence

You should install these dependence:
scales, ggsci, ggthemes, ggplot2, cowplot, ggrepel

install.packages(c("scales","ggsci","ggthemes","ggplot2","cowplot","ggrepel"))
